Programming the device:
The device comes preloaded with the code. To modify the range, follow this procedure: 
1. Unzip and install Arduino IDE with preloaded NewPing library.
2. Open Code > RangeDetectionDevice > RangeDetectionDevice.ino
3. Modify "upper_bound" and "lower_bound" global variables
4. Select Tools > Board > Arduino Nano
5. Select the correct COM port
6. Make sure the unit is OFF
7. Select Sketch > Upload
